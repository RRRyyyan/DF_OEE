# DF_OEE
## 2024.09.03 Updates
## DeviceInfo
### A. 未聯網
* 總時間=8:00AM~Now
* 取WIPLog紀錄
* 若為暫停&完工，且無前一筆時間戳記=>不記入
* 若為第一筆數據且為加工中狀態=>計算開工到當前時間
* 若不為第一筆且為加工中狀態=>不記入
### B. 聯網
* 總時間=8:00AM~Now
* 取UtilizationLog紀錄
* 取狀態為"RUN"的資料

## OEE計算方式
### A. 時間稼動率
計算每一天各機台最早時間戳記~最晚時間戳記 = A
計算每一天正常工時為8:00~17:00 & 17:30~20:30(加班) = B
結果=A/B*100%

### B. 性能稼動率
計算各機台實際加工時間 = C
結果=C/A*100%
